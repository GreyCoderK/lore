## How it works

1. **Commit** your code as usual
2. **Answer 3 questions** — Type, What, Why (90 seconds)
3. **Done** — A Markdown document captures the decision forever

## Why Lore?

Git tells you **who** changed **what** and **when**. But 6 months later, nobody remembers **why**. Lore captures that missing context at commit-time — when the knowledge is fresh.
