## Comment ça marche

1. **Committez** votre code comme d'habitude
2. **Répondez à 3 questions** — Type, Quoi, Pourquoi (90 secondes)
3. **C'est fait** — Un document Markdown capture la décision pour toujours

## Démarrage rapide

- [Installation](getting-started/installation.md) — Homebrew, Go, curl
- [Quickstart](getting-started/quickstart.md) — 5 minutes pour capturer votre premier "pourquoi"
- [Commandes](commands/index.md) — Référence complète

## Pourquoi Lore ?

Git vous dit **qui** a changé **quoi** et **quand**. Mais 6 mois plus tard, personne ne se souvient du **pourquoi**. Lore capture ce contexte manquant au moment du commit — quand la connaissance est encore fraîche.

Chaque commit contient de l'or. Lore l'extrait.
