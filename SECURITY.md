# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Lore, please report it responsibly.

**Do NOT open a public issue.**

Instead, send an email to: **leshodaimekossonou@gmail.com**

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You will receive an acknowledgment within 48 hours. We will work with you to understand and address the issue before any public disclosure.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest release | Yes |
| Older versions | No |
