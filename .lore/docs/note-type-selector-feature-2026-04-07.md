# Type selector feature

The type selector feature enhances user experience by providing an intuitive way to select types, surpassing traditional free-text input methods.

Instead of relying on manual typing or complex searches, arrow-key based type selection offers a seamless and efficient approach. This design choice benefits users in several ways:

*   **Reduced cognitive load**: By leveraging the familiar navigation of arrows, users can quickly switch between different types without needing to learn new keyboard shortcuts.
*   **Improved accuracy**: The direct control afforded by the arrow keys minimizes errors caused by typos or incorrect selections.
*   **Enhanced accessibility**: This feature facilitates better usability for individuals with disabilities, who may find it easier to navigate and select options using a keyboard.
